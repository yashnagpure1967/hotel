# 🏨 Team Alpha Hotel - Full-Stack Hotel Booking Website

A modern, responsive hotel booking website built using **Flask (Python)** for the backend and **HTML, CSS, JavaScript** for the frontend. This project is designed for hackathons and demonstrates a real-world application structure.

---

## 🚀 Features

### Frontend

* Home page with hero section and search form
* Rooms listing with dynamic data
* Room detail page with booking form
* Responsive design (mobile + desktop)
* Dynamic price calculation

### Backend

* Flask-based REST API
* SQLite database using SQLAlchemy
* Booking system with data storage
* Form validation and error handling
